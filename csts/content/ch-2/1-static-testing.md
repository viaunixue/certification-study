## 정적 분석

### 코딩 표준

* 초기화 하지 않고 사용한 코드
* 선언 후 사용하지 않은 함수 또는 변수
#### 코딩 스타일

* BSD

  ```java
  if(...)
  {
	  doSomething();
  }
  ```
* K&R

    ```java
    if(...){
        doSomething();
    }
    ```

* GNU

    ```java
    if(...)
        {
            doSomething();
        }
    ```
<br>

> MISRA-C : C 프로그래밍 언어 코딩 가이드라인

### 복잡도 분석

#### 순환 복잡도 = E (간선들의 개수) - N (노드들의 개수) + 2

* if, while, for, &&, || 를 만나면 +1
* switch 문의 각 case에 +1
